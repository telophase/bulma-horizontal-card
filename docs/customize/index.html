Customize Extension

How to customize the extension and use it in your Bulma project.

Install Bulma

If you don’t already have Bulma installed, it should have been installed when you installed bulma-horizontal-card via NPM as it is a dependency.

You can install Bulma manually with:

npm i bulma

Import into your SASS/CSS

After you’ve installed bulma-horizontal-card via NPM into your Bulma project directory, you can integrate it into your SCSS/SASS by adding the following line. Place this line after the line(s) that import Bulma (or at minimum, Bulma’s components/card.sass)

// scss
@import '@telophase/bulma-horizontal-card'

// sass
@import @telophase/bulma-horizontal-card

Unique to bulma-horizontal-card

$hcard-image-default-size Default width of the .card-image in horizontal cards 12.5%
